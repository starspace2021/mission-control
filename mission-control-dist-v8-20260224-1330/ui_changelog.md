# Mission Control UI 更新日志

## [v7.1.0] - 2026-02-24

### ✨ 新功能
- **全局搜索增强**: 完全重构搜索组件，支持键盘导航、分类标签、空状态提示
- **导航 Badge**: 侧边栏导航项显示实时数量（任务 5 个，记忆 3 个）
- **活动指示器**: 导航项添加平滑的活动状态指示器动画

### 🎨 视觉优化
- 热力图单元格悬停发光效果
- 标签云浮动动画增强
- 搜索弹窗添加快捷键提示栏

### ⚡ 性能优化
- 使用 motion.layoutId 实现平滑导航过渡
- CSS 动画使用 GPU 加速
