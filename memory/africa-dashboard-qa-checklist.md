# 非洲情报看板 - 质量检查规范

## 检查责任人
- **部门**: QA Agent (天津饭 - 七龙珠)
- **职责**: 每次迭代后全面检查网页功能

## 检查清单（每次迭代后必须执行）

### 1. 首页检查
- [ ] 页面正常加载，无404错误
- [ ] 简报列表显示正常
- [ ] 时间排序正确（最新在前）
- [ ] 导航栏显示正确（首页 | 非洲情报）

### 2. 二级页面检查（重点）
- [ ] **点击简报能进入详情页**
- [ ] 详情页内容完整显示
- [ ] 详情页格式正确（标题、时间、正文）
- [ ] 返回按钮/链接正常工作

### 3. 非洲情报列表页检查
- [ ] /africa 页面可访问
- [ ] 列表显示正常
- [ ] 筛选功能正常（如有）

### 4. 响应式检查
- [ ] 桌面端显示正常
- [ ] 移动端显示正常
- [ ] 无布局错乱

### 5. 性能检查
- [ ] 页面加载时间 < 3秒
- [ ] 图片正常加载
- [ ] 无控制台报错

## 检查记录
每次检查后记录到：
`memory/qa-reports/africa-dashboard/YYYY-MM-DD-HHMM.json`

## 发现问题处理
1. 记录问题详情
2. 通知 Engineering 部门修复
3. 修复后重新检查
4. 确认无误后再标记迭代完成
