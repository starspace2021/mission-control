# Mission Control UI 迭代日志

## v12.0 - 2026-02-24

### 设计方向
从 v11 的"丰富视觉"转向 v12 的"极简专业"风格，参考 Linear、Vercel Dashboard、Raycast 的设计理念。

### 主要改动

#### 1. 色彩系统重构
- **背景层级**: 引入 4 层表面色 (surface-0 到 surface-3)
- **强调色**: 降低饱和度，更克制专业
  - 蓝色: #3B82F6 → #4a7bff
  - 保持可读性同时减少视觉疲劳
- **文字层级**: 4 级文字色，更清晰的信息层级

#### 2. 间距系统标准化
- 引入 6 级间距变量 (4px 到 24px)
- 统一组件内边距和外边距
- 更紧凑的信息密度

#### 3. 组件系统简化
- **卡片**: 移除过度装饰，专注内容
- **按钮**: 3 种变体 (primary/secondary/ghost)
- **标签**: 6 种颜色变体，统一风格
- **输入框**: 更清晰的聚焦状态

#### 4. 新增组件
- `card-v12`: 极简卡片
- `glass-v12`: 克制玻璃态
- `btn-v12-*`: 按钮系统
- `input-v12`: 输入框
- `tag-v12-*`: 标签系统
- `stat-v12`: 统计卡片
- `progress-v12`: 进度条
- `nav-v12`: 导航系统
- `kanban-v12`: 看板系统
- `calendar-v12`: 日历系统
- `memory-v12`: 记忆卡片
- `empty-v12`: 空状态

#### 5. 动画优化
- 更克制的动画时长 (0.15s - 0.3s)
- 统一的缓动函数
- 支持减少动画偏好

#### 6. 响应式改进
- 看板: 4列 → 2列 → 1列
- 统计卡片: 自适应布局
- 移动端优化

### 文件变更
- 新增: `src/app/globals-v12.css`

### 下一步
- 将 v12 样式整合到主 globals.css
- 更新各页面组件使用新样式类
- 构建并部署
