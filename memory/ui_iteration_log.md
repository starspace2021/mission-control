# Mission Control UI 迭代日志

## v23.0 - 2026-02-27

### 优化内容

#### 1. 导航栏交互优化
- 添加 `layoutId` 动画背景切换效果
- 导航项添加悬停位移和点击缩放反馈
- 优化激活状态的视觉表现

#### 4. Dashboard 欢迎横幅
- 添加鼠标跟随光效

#### 5. 全局样式更新
- 更新版本号至 v23.0

- 优化动画过渡效果
- 新增微动效系统
