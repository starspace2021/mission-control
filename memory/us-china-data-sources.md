# 美国对华政策监控 - 数据源与监控清单

## 📋 6大政策类别监控清单

### 1. 贸易与关税 (Trade & Tariffs)

**核心监控点**:
- IEEPA (国际紧急经济权力法) 相关行动
- Section 122 (1974年贸易法第122条) 临时关税
- Section 301 (301条款调查)
- De Minimis 免税待遇
- 中美贸易谈判动态

**必查来源**:
- White House: https://www.whitehouse.gov/presidential-actions/
- USTR: https://ustr.gov/about-us/policy-offices/press-office/press-releases
- Commerce: https://www.commerce.gov/news/press-releases

**关键词**:
```
tariff, trade war, Section 301, USTR, IEEPA, import duty, trade deal, reciprocal trade, de minimis
```

---

### 2. 技术封锁与出口管制 (Tech & Export Controls)

**核心监控点**:
- BIS 实体清单 (Entity List) 更新
- 军事最终用户清单 (MEU List)
- 先进芯片/GPU/AI 芯片管制
- EDA 软件管制
- 华为、中兴等相关限制
- TikTok 数据安全

**必查来源**:
- BIS: https://www.bis.doc.gov/index.php/newsroom
- Federal Register: BIS 规则更新
- White House: 科技相关行政令

**关键词**:
```
export control, BIS, Entity List, MEU, semiconductor, chips, GPU, AI, EDA, Huawei, ZTE, TikTok, data security, advanced computing
```

---

### 3. 军事与地缘政治 (Military & Geopolitics)

**核心监控点**:
- INDOPACOM (印太司令部) 动态
- 台湾海峡军事行动
- 南海自由航行行动 (FONOP)
- 第七舰队活动
- 军事演习与威慑

**必查来源**:
- Defense: https://www.defense.gov/News/Releases/
- INDOPACOM: https://www.pacom.mil/Media/News/
- 第七舰队: https://www.c7f.navy.mil/Media/News/

**关键词**:
```
INDOPACOM, Pacific Fleet, Seventh Fleet, Taiwan Strait, South China Sea, FONOP, deterrence, military exercise, carrier strike group, ADIZ
```

---

### 4. 金融制裁与投资审查 (Sanctions & Investment Review)

**核心监控点**:
- OFAC SDN 清单更新
- CFIUS 审查案例
- 对外投资限制 (Outbound Investment)
- 对华投资禁令 (NS-CMIC List)
- 反洗钱/非法金融

**必查来源**:
- Treasury/OFAC: https://home.treasury.gov/policy-issues/financial-sanctions/recent-actions
- Treasury/CFIUS: https://home.treasury.gov/data/treasury-international-capital-ic-forms
- Federal Register: CFIUS 规则

**关键词**:
```
OFAC, SDN, sanction, CFIUS, investment restriction, outbound investment, NS-CMIC, money laundering, illicit finance, blocking sanction
```

---

### 5. 关键矿产与供应链 (Critical Minerals & Supply Chain)

**核心监控点**:
- 稀土出口管制
- 镓、锗、锑、钨等关键矿产
- 锂、钴、镍供应链
- 供应链安全审查
- 国防生产法案 (DPA) 动用

**必查来源**:
- Commerce: 供应链相关公告
- Defense Logistics Agency: https://www.dla.mil/
- USGS: 矿产相关报告

**关键词**:
```
rare earth, critical minerals, gallium, germanium, antimony, tungsten, lithium, cobalt, nickel, graphite, supply chain, export ban, DPA, Defense Production Act
```

---

### 6. 国会与立法动态 (Congress & Legislation)

**核心监控点**:
- Select Committee on the CCP (中国问题特别委员会)
- US-China Economic and Security Review Commission (USCC)
- 涉华法案进展
- 关键投票与听证
- 议员涉华言论

**必查来源**:
- Select Committee: https://selectcommitteeontheccp.house.gov/news
- USCC: https://www.uscc.gov/research-and-analysis
- Congress.gov: 涉华法案搜索
- 关键议员: Marco Rubio, John Moolenaar, Raja Krishnamoorthi

**关键词**:
```
Select Committee on CCP, USCC, congressional hearing, China bill, legislation, Marco Rubio, Moolenaar, Krishnamoorthi
```
