# 非洲安全事件主动预警系统

## 监控范围

### P0 级监控（即时告警）
- **尼日利亚**
  - 中资企业安全事件
  - 武装袭击、绑架事件
  - 关键词: 中国工人、中资企业、绑架、袭击
  
- **刚果（金）**
  - 东部武装冲突
  - 中资企业影响评估
  - 关键词: M23、ADF、中资企业、撤离

- **埃塞俄比亚**
  - 提格雷地区冲突
  - 中国投资项目影响
  - 关键词: 提格雷、中国投资、冲突

### P1 级监控（日报汇总）
- 南非：排外事件、中资企业抗议
- 赞比亚：矿业冲突、劳工问题
- 津巴布韦：政治动荡、投资政策变化

## 预警触发条件

### 即时告警（飞书消息）
- 中资企业人员伤亡
- 中资企业设施遭袭
- 政府发布紧急撤离建议
- 关键词 + 地点 + 时间 三重匹配

### 日报标注
- 一般性安全事件
- 潜在风险提示
- 地区安全趋势变化

## 数据源

- Daily Trust Nigeria (安全版块)
- Congo News Agency
- Addis Standard (埃塞俄比亚)
- 国际媒体: Reuters, BBC Africa

## 自动化流程

```
每2小时检查一次:
1. 搜索安全相关新闻
2. 关键词匹配
3. 涉华相关性评估
4. P0级事件 → 即时飞书告警
5. 其他事件 → 加入日报队列
```
