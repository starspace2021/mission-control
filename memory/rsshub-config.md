# RSSHub 配置（备用方案）

## 官方实例
- https://rsshub.app/twitter/user/{username}

## 备用实例
- https://rsshub.rssforever.com/twitter/user/{username}
- https://rsshub.pseudoyu.com/twitter/user/{username}

## 使用格式
```
https://rsshub.app/twitter/user/StateDept
https://rsshub.app/twitter/user/WhiteHouse
https://rsshub.app/twitter/user/DeptofDefense
```
