## 🔴 金融工程 (P0)

### 1. Robust Hedging of Path-Dependent Options Using Min-Max Algorithm

- **来源:** arXiv q-fin (2025)
- **作者:** Purba Banerjee, Srikanth Iyer, Shashi Jain
- **摘要:** 研究使用min-max算法对路径依赖型期权（如回望期权、亚式期权）进行稳健对冲，计算稳健价格边界。
- **关键词命中:** derivatives pricing, options, hedging
- **关联度:** ⭐⭐⭐⭐⭐ 直接涉及期权定价与对冲策略

### 2. GARCH Models for Volatility Forecasting in High-Frequency Trading

- **来源:** ResearchGate (2025-02-21)
- **摘要:** 探索GARCH模型在高频交易(HFT)环境中的波动率预测应用，分析2020-2025年数据。
- **关键词命中:** GARCH, volatility modeling, high-frequency trading
- **关联度:** ⭐⭐⭐⭐⭐ 完美结合波动率建模与高频交易

### 3. Reinforcement Learning for Quantitative Trading

- **来源:** ACM Digital Library
- **摘要:** 综述强化学习在量化交易中的应用，通过与环境交互的试错学习方法优化交易策略。
- **关键词命中:** algorithmic trading, reinforcement learning, quantitative finance
- **关联度:** ⭐⭐⭐⭐ 前沿的AI+金融交叉研究

### 4. How Hedge Funds Use Machine Learning for Derivatives

- **来源:** Substack/研究报告 (2025)
- **摘要:** 2025年对Petrobras期权的研究显示，深度学习模型相比Black-Scholes模型实现了64.3%的均方误差降低。
- **关键词命中:** machine learning, derivatives, options pricing
- **关联度:** ⭐⭐⭐⭐ 展示ML在衍生品定价中的实际效果

### 5. GARCH Option Pricing with Volatility Derivatives

- **来源:** ScienceDirect
- **摘要:** 研究联合估计对GARCH期权定价的好处，利用股票收益和波动率衍生品进行联合估计。
- **关键词命中:** GARCH, option pricing, volatility derivatives
- **关联度:** ⭐⭐⭐⭐ 波动率衍生品定价的前沿方法
